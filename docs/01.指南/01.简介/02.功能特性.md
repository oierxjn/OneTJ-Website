## 功能概览

OneTJ 致力于为同济大学学生提供一个简洁、高效的信息查询平台。

## 核心功能

### 学生信息查询

通过同济大学官方 API 获取学生的完整学籍信息，包括：

| 信息类别 | 内容 |
|---------|------|
| **基本信息** | 学号、姓名、性别 |
| **学院信息** | 学院代码、学院名称、专业代码、专业名称 |
| **学籍信息** | 入学日期、预计毕业日期、学制、当前年级 |
| **培养层次** | 培养类别、培养层次 |
| **学籍状态** | 学籍状态、注册状态、是否在校 |
| **政治面貌** | 政治面貌代码、政治面貌名称 |
| **导师信息** | 辅导员工号、副辅导员工号 |

### 课程表展示

- **完整课表**：展示整学期的课程安排
- **详细信息**：课程名称、教室、授课教师
- **时间信息**：星期、节次（开始-结束）
- **智能排序**：自动按星期和节次排序
- **索引系统**：支持按条件快速查找课程

### 校历查看

- **学期名称**：如"2024-2025 学年第一学期"
- **当前周次**：自动计算并显示当前是第几周
- **学期范围**：显示学期的开始和结束日期
- **总周数**：显示学期的总周数

## 技术特性

### 跨平台支持

OneTJ 使用 Flutter 框架，支持以下平台：

::: code-group

```sh [移动端]
- Android 5.0+
- iOS 12.0+
```

```sh [桌面端]
- Windows 10+
- macOS 10.14+
- Linux (主流发行版)
```

```sh [其他]
- Web (现代浏览器)
- OpenHarmony (鸿蒙)
```

:::

### 本地缓存

采用 Hive 作为本地数据库：

- **高性能**：Hive 是轻量级、高性能的 NoSQL 数据库
- **离线访问**：缓存数据后可离线查看
- **自动刷新**：启动时自动获取最新数据

### OAuth2 认证

- **官方认证**：使用同济大学官方 Keycloak 系统
- **安全可靠**：采用标准的 OAuth2 授权码模式
- **自动刷新**：Access Token 过期时自动使用 Refresh Token 刷新

### 国际化

- 支持中文、英文切换
- 使用 ARB 格式管理翻译资源
- 可通过应用内设置动态切换语言

```

└──────┬──────┘
       │
       ↓
┌─────────────┐
│  ViewModel  │  视图逻辑

       ↓

└─────────────┘
```

### 依赖注入

通过 Repository 模式管理数据层：

- **TokenRepository**：管理访问令牌
- **StudentInfoRepository**：管理学生信息缓存
- **SchoolCalendarRepository**：管理校历缓存
- **CourseScheduleRepository**：管理课程表缓存

### 路由管理

使用 go_router 进行声明式路由：

- 类型安全的路由导航
- 深度链接支持
- 自定义过渡动画
